## Intended Database Users
The primary users of this database are airline operation analysts and airport management. The database serves to improve traffic efficiency by minimalizing contributing factors to flight delays.
Additional users are travelers looking for trends within flight delays and researchers studying data on flight operations.

## Revision of Database Design
Upon analysis of the proposed database design, it demonstrated its relations satisfied BCNF. None of its relations were shown to create redundancies in data with duplicates, and all determinants in functional dependencies were found to be super keys. Attributes within each entity table were only related to the entity at hand, further preventing data redundancy within the database.
